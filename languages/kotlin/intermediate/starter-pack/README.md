# Kotlin Intermediate Starter Pack

This folder provides a minimal starting point for Kotlin intermediate modules.

## How to use
1. Copy this folder into a new working directory.
2. Rename it for the module you are working on.
3. Expand the sample entry point and add tests as you complete the walkthrough.

## Quick verification
- Run the module tests (if configured).
- Confirm the starter entry point runs without errors.
- Example command: `kotlinc Main.kt -include-runtime -d main.jar && java -jar main.jar`
