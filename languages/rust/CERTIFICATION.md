# Rust Certification Guide

This guide defines the requirements and evidence for Rust certifications.

## Certified Beginner

### Required modules
- [Core Concepts (Intermediate)](intermediate/modules/core-concepts.md)

### Required projects
- [CLI Utility Project](intermediate/projects/cli-utility.md)

### Evidence required
- Tests or validation logs
- README with setup and usage
- Screenshots or logs when relevant

### Minimum rubric threshold
- Meets in all criteria
- Exceeds in at least 2 criteria
- [Beginner Rubric](../../assessments/rubrics/beginner-rubric.md)
