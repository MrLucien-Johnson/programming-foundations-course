# AI Certification

Note: Module and project links point to the planned AI path in
`languages/ai/` and will be created in a later task.

## Certification levels
### Certified Beginner
- Complete Modules 01-03.
- Complete Mini Project 1.
- Provide README and basic test or run evidence.

### Certified Intermediate
- Complete Modules 01-06.
- Complete Mini Projects 1 and 2.
- Show performance basics and security hygiene awareness.

### Certified Advanced
- Complete Modules 01-07.
- Complete the capstone project.
- Provide system design notes and profiling evidence.

## Evidence required
- Tests passing, or a clear manual test log when tests are not provided.
- A concise `README.md` with setup and run steps.
- Screenshots or logs that demonstrate key features.
- Deployment proof when the project includes a deployable component.

## Minimum rubric scores
- Beginner: "Meets" in Code Review and Testing.
- Intermediate: "Meets" in Code Review, Testing, and Security.
- Advanced: "Meets" in all rubrics and "Exceeds" in at least 2 criteria.

Rubrics: [Code Review](../../assessments/rubrics/code-review-rubric.md),
[Testing](../../assessments/rubrics/testing-rubric.md),
[Security](../../assessments/rubrics/security-rubric.md),
[System Design](../../assessments/rubrics/system-design-rubric.md).

## Retake and resubmission guidance
- One resubmission per attempt is allowed.
- Address every reviewer note and summarize changes in the PR.
- If still below standard, start a new submission cycle.
