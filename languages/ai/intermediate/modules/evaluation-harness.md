# Evaluation Harness (Intermediate)

Design repeatable evaluations for AI outputs.

## Topics
- test cases and golden sets
- automated scoring or rubric checks
- regression tracking

## Deliverables
- evaluation harness or scripts
- evaluation report with findings

## Guided Walkthrough



Follow these steps to turn the lesson into a real, working deliverable.

1. Copy the starter pack from `languages/ai/intermediate/starter-pack` into a new working folder.
2. Review the module goals and plan how you will build an evaluation harness for repeatable testing.
3. Define the evaluation categories and success criteria.
4. Automate evaluation runs and capture metrics.
5. Summarize results and prioritize fixes.
6. Document decisions in a short README section (assumptions, tradeoffs, next steps).

## Starter Pack
