# Prompting Basics Quiz Answers

## Question 1: Which approach makes a prompt clearer?
**Answer: A** - Use specific instructions and examples

**Explanation:** Specific instructions reduce ambiguity.

---

## Question 2: If a prompt is not working well, you should:
**Answer: A** - Change one variable at a time

**Explanation:** Small, focused changes help you see what improves results.

---

## Question 3: A useful prompt structure includes:
**Answer: A** - Goal, constraints, and output format

**Explanation:** Structure keeps the model on track.
