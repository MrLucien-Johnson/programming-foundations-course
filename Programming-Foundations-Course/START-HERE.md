# Welcome to Programming Foundations!

## 🎉 Congratulations on Starting Your Journey

You're about to learn programming from the ground up. Whether you're looking to change careers, advance in your current role, or simply build new skills, this course is designed to take you from complete beginner to someone who can build real applications.

## 📚 What This Course Is

**Programming Foundations: From Zero to Portfolio** is a comprehensive, beginner-friendly course that teaches you:

- **Python programming** - One of the most popular and beginner-friendly languages
- **C# programming** - A powerful language used in professional software development
- **Real-world skills** - You'll build actual applications, not just toy examples
- **Portfolio projects** - Complete applications you can show to employers

This course combines two complete programming workbooks:
- A Python Beginner Workbook (7 modules)
- A C# Beginner Workbook (7 modules)

Each workbook takes you from "Hello, World!" to building a complete Task Tracker application that you can add to your portfolio.

## 👥 Who This Course Is For

This course is perfect for:

- **Complete beginners** - Never written a line of code? Perfect! We start from absolute zero.
- **Career changers** - Looking to transition into tech? This gives you real skills and portfolio projects.
- **Career advancers** - Want to stand out in your current role? Programming skills open doors.
- **Self-learners** - Prefer to learn at your own pace? This course is self-paced and flexible.
- **Confidence builders** - Nervous about coding? We use encouraging, clear language throughout.
- **Practical learners** - Learn best by doing? Every module has hands-on exercises.

**Age:** Adults and older teens (16+) who want practical, job-relevant skills.

## ❌ Who This Course Is NOT For

This course might not be right for you if:

- **You're already an experienced programmer** - This is designed for beginners.
- **You want advanced topics** - We cover fundamentals, not advanced algorithms or frameworks.
- **You need live instruction** - This is self-paced with written materials and exercises.
- **You want a quick "coding bootcamp"** - This is thorough and takes time to complete properly.
- **You're looking for a magic solution** - Learning to code requires practice and effort.

**Honest note:** If you're already comfortable with variables, loops, functions, and classes, you might find this too basic. Consider intermediate courses instead.
