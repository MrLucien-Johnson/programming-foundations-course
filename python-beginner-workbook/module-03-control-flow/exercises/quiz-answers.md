# Module 3 Quiz Answers

1. **B** - `==` (double equals for comparison)
2. **C** - Infinite loop (if counter never updates)
3. **B** - Indentation (Python uses spaces, not braces)
4. **A** - "Else if" - check another condition
5. **True** - for loops run a specific number of times (or for each item)

## Key Takeaways

- Use `==` for comparison, `=` for assignment
- Always update counters in while loops
- Python uses indentation for code structure
- `elif` checks another condition if previous ones are false
- for loops iterate a specific number of times or through collections
