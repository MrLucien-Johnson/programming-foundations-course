# Exercise 3: Task Reminder Loop

## 🎯 Your Task

Create a program that loops through a list of tasks and displays reminders using a for loop.

## 📝 Steps

1. Create a new file called `task_reminder.py`

2. Write code that:
   - Creates a list of tasks (at least 3 tasks)
   - Uses a for loop to go through each task
   - Displays each task as a reminder
   - Numbers the tasks (1, 2, 3...)

## 💡 Example Output

```
=== Task Reminders ===
1. Buy groceries
2. Finish report
3. Call client
4. Review documents

All tasks displayed!
```

## ✅ Tips

- Create a list: `tasks = ["Task 1", "Task 2", ...]`
- Use `enumerate()` to get both index and task: `for i, task in enumerate(tasks, start=1):`
- Or use `range()` with list length
- Display each task with its number

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Creating lists
- Using for loops
- Looping through collections
- Using enumerate() to number items
