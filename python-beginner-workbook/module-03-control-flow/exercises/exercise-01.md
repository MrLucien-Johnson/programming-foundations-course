# Exercise 1: Grade Classifier

## 🎯 Your Task

Create a program that classifies student grades using if/elif/else statements.

## 📝 Steps

1. Create a new file called `grade_classifier.py`

2. Write code that:
   - Asks for a student's score (0-100)
   - Classifies the grade:
     - A: 90-100
     - B: 80-89
     - C: 70-79
     - D: 60-69
     - F: Below 60
   - Displays the grade

## 💡 Example Output

```
Enter student score: 85
Grade: B
```

## ✅ Tips

- Use `int()` or `float()` to convert input to number
- Use `elif` for multiple conditions
- Remember: `>=` means "greater than or equal to"
- Test with different scores to make sure it works

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Using if/elif/else statements
- Using comparison operators
- Making decisions based on numeric ranges
