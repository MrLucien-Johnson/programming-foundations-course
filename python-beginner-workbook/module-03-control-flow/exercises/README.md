# Module 3 Exercises

Welcome to the exercises section! This is where you'll practice what you've learned and build confidence.

## 📚 How to Use These Exercises

### Step 1: Read the Lesson First
Before trying exercises, make sure you've read through `README.md` in this module. Understanding the concepts first makes exercises much easier!

### Step 2: Try the Exercises
- **exercise-01.md** - Grade Classifier
- **exercise-02.md** - Login Attempt Limit
- **exercise-03.md** - Task Reminder Loop

Each exercise file contains:
- Clear task description
- Step-by-step instructions
- Example output
- Tips to help you succeed

### Step 3: Check Solutions
After you've tried an exercise (or if you get stuck), check the solution file:
- **exercise-01-solution.md** - Solution for Exercise 1
- **exercise-02-solution.md** - Solution for Exercise 2
- **exercise-03-solution.md** - Solution for Exercise 3

**Important:** Try the exercise yourself first! Looking at solutions too early prevents learning.

### Step 4: Try the Challenges
- **debug-challenge.md** - Find and fix errors in broken code
- **quiz.md** - Test your understanding with questions

Then check:
- **debug-challenge-solution.md** - See how to fix the errors
- **quiz-answers.md** - Check your quiz answers

## 💡 Tips for Success

1. **Type code yourself** - Don't copy-paste! Typing helps you learn
2. **Watch your indentation** - Python is very picky about spaces!
3. **Test your loops** - Make sure they end (no infinite loops!)
4. **Use print() to debug** - See what's happening inside your code
5. **Take breaks** - Learning takes time, don't rush

## 🎯 Exercise Order

We recommend doing exercises in this order:
1. Exercise 1 (Grade Classifier) - Practices if/elif/else
2. Exercise 2 (Login Limit) - Practices while loops
3. Exercise 3 (Task Loop) - Practices for loops
4. Debug Challenge - Practices finding errors
5. Quiz - Tests overall understanding

## ✅ What Success Looks Like

You'll know you're ready to move on when:
- ✅ You can write if/elif/else statements correctly
- ✅ You can create for loops
- ✅ You can create while loops without infinite loops
- ✅ Your indentation is correct
- ✅ Your code runs without errors

## 🚀 Remember

- **Indentation matters!** - Python uses spaces to show structure
- **Watch for infinite loops** - Always update counters in while loops
- **Use == for comparison** - Not = (that's for assignment)
- **Practice makes perfect** - The more you code, the easier it gets
