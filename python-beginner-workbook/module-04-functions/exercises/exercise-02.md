# Exercise 2: Discount Calculation Function

Create a function that calculates discounted price. Function should take original price and discount percentage, return final price.

**Requirements:**
- Function name: `calculate_discount`
- Parameters: `price` (float), `discount_percent` (float)
- Return: Final price after discount

**Example:**
```python
final = calculate_discount(100.00, 15)  # 15% discount
# Should return 85.00
```
