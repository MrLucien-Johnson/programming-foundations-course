# Exercise 1: Print Header Function

Create a function that prints a formatted header. Function should take a title as parameter and display it with decorative lines.

**Requirements:**
- Function name: `print_header`
- Parameter: `title` (string)
- Display title with decorative lines above and below

**Example output:**
```
========================================
Order Summary
========================================
```

See `exercise-01-solution.md` for solution.
