# Exercise 3: Password Validation Function

Create a function that validates password length. Function should check if password is at least 8 characters.

**Requirements:**
- Function name: `is_valid_password`
- Parameter: `password` (string)
- Return: `True` if password is 8+ characters, `False` otherwise

**Example:**
```python
if is_valid_password("mypassword"):
    print("Valid!")
```
