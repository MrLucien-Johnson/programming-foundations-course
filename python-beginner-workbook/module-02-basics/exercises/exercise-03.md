# Exercise 3: Weekly Pay Estimator

## 🎯 Your Task

Create a calculator that estimates weekly gross pay from hours worked and hourly rate.

## 📝 Steps

1. Create a new file called `pay_estimator.py`

2. Write code that:
   - Asks for hours worked this week
   - Asks for hourly rate
   - Calculates gross pay (hours × rate)
   - Displays a summary with all the information

## 💡 Example Output

```
Enter hours worked this week: 40
Enter hourly rate: £15.50

=== Weekly Pay Estimate ===
Hours Worked: 40.0
Hourly Rate: £15.50
Gross Pay: £620.00
```

## ✅ Tips

- Use `float(input())` for both hours and rate (allows decimals)
- Gross pay = hours × rate
- Use f-strings to format your output nicely
- Use `:.2f` to show 2 decimal places for currency
- Save and run: `python pay_estimator.py`

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Getting multiple inputs
- Performing multiplication
- Formatting numeric output
- Creating a useful calculator
