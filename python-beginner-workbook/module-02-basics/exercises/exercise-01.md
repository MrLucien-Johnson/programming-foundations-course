# Exercise 1: Name & Age Summary

## 🎯 Your Task

Create a program that asks for a user's name and age, then displays a friendly summary.

## 📝 Steps

1. Create a new file called `name_age_summary.py`

2. Write code that:
   - Asks for the user's name
   - Asks for the user's age (as a number)
   - Displays a friendly message with their name
   - Shows their current age
   - Calculates and shows their age in 10 years

## 💡 Example Output

```
Enter your name: Sarah
Enter your age: 25

Hello, Sarah!
You are currently 25 years old.
In 10 years, you'll be 35 years old.
```

## ✅ Tips

- Use `input()` to get the name (returns a string)
- Use `int(input())` to get the age (converts to integer)
- Use f-strings to format your output: `f"Hello, {name}!"`
- Remember to convert age to integer before doing math
- Save your file and run: `python name_age_summary.py`

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Using `input()` to get user data
- Converting input types (`int()`)
- Storing data in variables
- Using f-strings for formatting
- Doing simple calculations
