# Debug Challenge: Find the Errors

## 🎯 Your Task

Here's some broken Python code. Can you find and fix all the errors?

## 🐛 The Broken Code

```python
name = input("Enter your name: )
age = input("Enter your age: ")
total_age = age + 10
print(f"Hello {name}, in 10 years you'll be {total_age}")

price = input("Enter price: ")
vat = price * 0.20
total = price + vat
print(f"Total: £{total}")
```

## 📝 Steps

1. **Read the code carefully**

2. **Find all the mistakes:**
   - Error 1: _______________
   - Error 2: _______________
   - Error 3: _______________
   - Error 4: _______________

3. **Write the corrected version**

4. **Test it** - Create a file, paste your fixed code, and run it

## 💡 Hints

- Check all the quotes - are they matched?
- What type does `input()` return?
- Can you do math with strings?
- What happens when you try to multiply a string by a number?

## 🎓 Learning Goals

By completing this challenge, you'll practice:
- Finding syntax errors (missing quotes)
- Understanding input types
- Converting strings to numbers
- Fixing type errors

## ✅ Success Criteria

Your fixed code should:
- ✅ Run without errors
- ✅ Correctly handle name input
- ✅ Correctly calculate age + 10
- ✅ Correctly calculate VAT and total
