# Module 1 Quiz Answers

## Question 1: What does `print()` do?
**Answer: B** - Displays text on the screen

**Explanation:** `print()` is a built-in Python function that outputs text to the console/terminal. It's one of the most basic and commonly used functions in Python.

**Example:**
```python
print("Hello")  # Displays: Hello
```

---

## Question 2: What file extension do Python files use?
**Answer: B** - `.py`

**Explanation:** Python files must end with `.py` extension. This tells Python (and your editor) that the file contains Python code.

**Example:**
- `hello.py` ✅ Correct
- `hello.txt` ❌ Wrong
- `hello.python` ❌ Wrong

---

## Question 3: How do you run a Python file called `hello.py`?
**Answer: B** - `python hello.py`

**Explanation:** To run a Python file, you use the `python` command followed by the filename. On Mac/Linux, you might need to use `python3` instead.

**Example:**
```bash
python hello.py        # Windows
python3 hello.py      # Mac/Linux (sometimes)
```

---

## Question 4: True or False: Python needs semicolons at the end of each line.
**Answer: False**

**Explanation:** Python does NOT require semicolons. It uses line breaks to separate statements. This makes Python code cleaner and easier to read than some other languages.

**Example:**
```python
print("Hello")  # ✅ No semicolon needed
print("World")  # ✅ Python uses line breaks instead
```

---

## Question 5: What is Python?
**Answer: B** - A programming language

**Explanation:** Python is a high-level programming language used to write software, scripts, and applications. It's not a code editor (that would be Cursor or VS Code) or an operating system.

**Key facts about Python:**
- Created by Guido van Rossum
- Named after Monty Python (the comedy group)
- Easy to learn and read
- Very popular and widely used
