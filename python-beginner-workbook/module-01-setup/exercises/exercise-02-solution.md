# Exercise 2 Solution: Multi-Line Message

## ✅ Sample Solution

Here's one way to solve this exercise:

```python
print("Today is a great day to learn Python!")
print("I'm excited to start this journey.")
print("Let's build something amazing together!")
```

## 📝 Explanation

- **Line 1:** First message about learning Python
- **Line 2:** Second message expressing excitement
- **Line 3:** Third message about building things

## 💡 Important Points

- Each `print()` statement automatically creates a new line
- You can have as many `print()` statements as you want
- The order matters - Python executes from top to bottom

## 🎯 Your Solution vs This Solution

**Remember:** Your story can be completely different! As long as:
- You have at least 3 `print()` statements
- Each line displays correctly
- Your program runs without errors
