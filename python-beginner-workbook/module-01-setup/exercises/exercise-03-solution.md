# Exercise 3 Solution: Predict the Output

## ✅ The Output

When you run this code:

```python
print("First line")
print("Second line")
print("Third line")
```

**The output will be:**

```
First line
Second line
Third line
```

## 📝 Explanation

- Each `print()` statement displays its text on a new line
- Python executes code from top to bottom
- The output appears in the same order as the code

## 💡 Why This Matters

Understanding what code does before running it helps you:
- Debug errors faster
- Write better code
- Think like a programmer
- Predict program behavior
