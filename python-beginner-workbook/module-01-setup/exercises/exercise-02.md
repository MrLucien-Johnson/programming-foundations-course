# Exercise 2: Multi-Line Message

## 🎯 Your Task

Create a program that prints a short story or message across multiple lines.

## 📝 Steps

1. Create a new file called `story.py`

2. Write at least 3 `print()` statements
   - Each should print a different line
   - Make it meaningful to you

## 💡 Example Output

```
Today is a great day to learn Python!
I'm excited to start this journey.
Let's build something amazing together!
```

## ✅ Tips

- Use multiple `print()` statements
- Each `print()` creates a new line automatically
- Be creative! Make it your own
- Save and run: `python story.py`

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Using multiple print statements
- Understanding how Python displays output
- Organizing code logically
