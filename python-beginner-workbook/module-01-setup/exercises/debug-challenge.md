# Debug Challenge: Find the Errors

## 🎯 Your Task

Here's some broken Python code. Can you find and fix all the errors?

## 🐛 The Broken Code

```python
print("Hello World)
print("This is Python")
print('I'm learning to code')
```

## 📝 Steps

1. **Read the code carefully**

2. **Find all the mistakes:**
   - Error 1: _______________
   - Error 2: _______________
   - Error 3: _______________

3. **Write the corrected version**

4. **Test it** - Create a file, paste your fixed code, and run it

## 💡 Hints

- Check all the quotes - are they matched?
- Look at each line carefully
- Python is picky about quotes!

## 🎓 Learning Goals

By completing this challenge, you'll practice:
- Finding syntax errors
- Understanding Python's quote rules
- Fixing broken code

## ✅ Success Criteria

Your fixed code should:
- ✅ Run without errors
- ✅ Display all three messages correctly
- ✅ Use proper quote matching
