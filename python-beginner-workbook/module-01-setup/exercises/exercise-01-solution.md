# Exercise 1 Solution: Personal Greeting Program

## ✅ Sample Solution

Here's one way to solve this exercise:

```python
print("Hello! I'm Sarah.")
print("I love reading science fiction books.")
print("I hope to learn how to build useful tools with code.")
```

## 📝 Explanation

- **Line 1:** Uses `print()` to display a greeting with your name
- **Line 2:** Uses `print()` to share a hobby or interest
- **Line 3:** Uses `print()` to express a learning goal

## 💡 Important Points

- Each `print()` statement creates a new line automatically
- Text must be in quotes (single `'` or double `"` quotes work)
- You can use your own name and information - make it personal!
