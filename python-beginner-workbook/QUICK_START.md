# Quick Start: Your First Python Program in 5 Minutes

This guide gets you running your first Python program quickly. For detailed instructions, see Module 1.

## ⚡ Step 1: Install Python

1. Go to https://www.python.org/downloads/
2. Download Python 3.11 or newer (latest stable version)
3. **Important:** Check "Add Python to PATH" during installation (Windows)
4. Run the installer and follow the steps
5. Open a terminal and verify: `python --version`
6. You should see a version number ✅

## ⚡ Step 2: Install Cursor or VS Code

**Option A: Cursor** (Recommended)
- Download from https://cursor.sh
- Install and open Cursor

**Option B: Visual Studio Code**
- Download from https://code.visualstudio.com/
- Install and open VS Code

Then install the **Python** extension:
- Click Extensions icon (left sidebar)
- Search "Python" (by Microsoft)
- Click Install

## ⚡ Step 3: Create and Run Your First Program

1. **Open Cursor or VS Code**

2. **Open the integrated terminal:**
   - Press `` Ctrl + ` `` (backtick key)
   - Or: `Terminal > New Terminal`

3. **Create a folder and file:**
   ```bash
   mkdir hello-world
   cd hello-world
   ```

4. **Create a file called `hello.py`:**
   - Click "New File" or press `Ctrl + N` (Windows) / `Cmd + N` (Mac)
   - Type: `print("Hello, World!")`
   - Save as `hello.py` (`Ctrl + S` or `Cmd + S`)

5. **Run it:**
   ```bash
   python hello.py
   ```

6. **You should see:** `Hello, World!` ✅

## 🎉 Success!

You just created and ran your first Python program!

**Troubleshooting:**
- If `python` isn't found: Make sure Python is added to PATH, or try `python3` instead
- If terminal doesn't open: Use `View > Terminal` menu
- On Mac/Linux, you might need to use `python3` instead of `python`
