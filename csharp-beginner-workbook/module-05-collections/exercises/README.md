## 🎯 Key Concepts to Practice

- **Arrays** - Fixed-size collections
- **Lists** - Dynamic collections that grow/shrink
- **Indexes** - Position numbers (start at 0!)
- **Adding items** - `list.Add("item")`
- **Removing items** - `list.Remove("item")`
- **Counting items** - `.Length` for arrays, `.Count` for lists
- **For loops** - Loop by index
- **Foreach loops** - Loop through each item (easier!)
