## 🎯 How to Use These Files

1. **Read the module lesson first** - Make sure you've read through `module-06-oop-intro/README.md` and understand the concepts

2. **Start with the exercise files** (exercise-01.md, exercise-02.md, etc.)
   - Read the task description carefully
   - Try to solve it yourself
   - Write your code in a new project
   - Test it with `dotnet run`

3. **Then check the solution files** (exercise-01-solution.md, etc.)
   - Compare your approach
   - Learn from differences
   - See alternative solutions
   - Understand the explanations

4. **Important:** Don't peek at solutions until you've tried!

## 📝 Creating Your Projects

For each exercise, create a new project:

```bash
dotnet new console -o ProjectName
cd ProjectName
```

Then open `Program.cs` in Cursor or VS Code and write your code.

**Important:** When working with classes, you can define the class in the same `Program.cs` file, or create a separate file. For these exercises, putting everything in `Program.cs` is fine.

## 🚀 Exercise Order

We recommend doing the exercises in order:
1. Exercise 1 (TaskItem Class) - Basic class with properties
2. Exercise 2 (Product Class) - Class with constructor
3. Exercise 3 (Customer Class) - Class with methods
4. Debug Challenge - Finding and fixing errors
5. Quiz - Testing your understanding

## 🎯 Key Concepts to Practice

- **Class definition** - `class ClassName { ... }`
- **Properties** - `public string Name { get; set; }`
- **Constructors** - `public ClassName(parameters) { ... }`
- **Creating objects** - `ClassName obj = new ClassName();`
- **Methods in classes** - `public void MethodName() { ... }`
- **Using objects** - Setting properties, calling methods
