## 🎯 How to Use These Files

1. **Read the module lesson first** - Make sure you've read through `module-02-basics/README.md` and understand the concepts

2. **Start with the exercise files** (exercise-01.md, exercise-02.md, etc.)
   - Read the task description carefully
   - Try to solve it yourself
   - Write your code in a new project
   - Test it with `dotnet run`

3. **Then check the solution files** (exercise-01-solution.md, etc.)
   - Compare your approach
   - Learn from differences
   - See alternative solutions
   - Understand the explanations

4. **Important:** Don't peek at solutions until you've tried!

## 💡 Tips for Success

- **Read carefully** - Make sure you understand what's being asked
- **Type code yourself** - Don't copy-paste! Typing helps you learn
- **Test your code** - Run it and see what happens
- **Make mistakes** - Errors teach you how things work
- **Compare solutions** - See different ways to solve the same problem
- **Experiment** - Try variations and improvements

## 🎓 Learning Approach

1. Read the module lesson (`README.md`)
2. Read the exercise description
3. Think about what you need to do
4. Write your code
5. Test it (`dotnet run`)
6. Fix any errors
7. Compare with the solution
8. Try variations and improvements

## 📝 Creating Your Projects

For each exercise, create a new project:

```bash
dotnet new console -o ProjectName
cd ProjectName
```

Then open `Program.cs` in Cursor or VS Code and write your code.

## ✅ Remember

- There are many ways to solve problems
- If your code works, you've succeeded!
- Focus on understanding concepts, not memorizing code
- Making mistakes is normal and helpful for learning
- Each exercise builds on the previous ones

## 🚀 Exercise Order

We recommend doing the exercises in order:
1. Exercise 1 (Name & Age) - Basic input and output
2. Exercise 2 (VAT Calculator) - Calculations and formatting
3. Exercise 3 (Pay Estimator) - More complex calculations
4. Debug Challenge - Finding and fixing errors
5. Quiz - Testing your understanding

Good luck with your learning! 🎉
