# Module 7 Exercises and Enhancements

This folder contains enhancement challenges and quizzes for your Task Tracker project.

### Solutions

## 🎯 How to Use These Files

1. **Build the basic version first** - Make sure your Task Tracker works with the core features:
   - Add tasks
   - View tasks
   - Complete tasks
   - Exit

2. **Then try enhancements** - Once the basic version works, try adding enhancements one at a time:
   - Read the enhancement description
   - Think about how to implement it
   - Modify your code
   - Test it thoroughly

3. **Check solutions** - If you get stuck, look at the solutions folder for guidance

4. **Take the quiz** - Test your understanding of what you built

## 💡 Tips for Enhancements

- **Work on one at a time** - Don't try to add everything at once
- **Test after each change** - Make sure it still works
- **Back up your code** - Save working versions before making changes
- **Read carefully** - Understand what each enhancement requires
- **Experiment** - Try variations and improvements

## 🎓 Enhancement Order

We recommend trying enhancements in this order:
1. Enhancement 3 (Delete Tasks) - Simplest addition
2. Enhancement 2 (Categories) - Adds useful organization
3. Enhancement 1 (Due Dates) - Adds time-based features
4. Enhancement 4 (Colors) - Improves user experience
5. Enhancement 5 (Save/Load) - Most advanced, adds persistence

## ✅ Remember

- Enhancements are optional - your basic project is already complete!
- Each enhancement makes your project more impressive
- Don't worry if some are challenging - they're meant to stretch your skills
- Focus on understanding, not just getting it to work
