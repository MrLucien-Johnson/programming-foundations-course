# Debug Challenge: Find the Errors

## 🎯 Your Task

Look at this broken code and find **all 3 errors**. Fix them so the program runs correctly!

## 🐛 The Broken Code

```csharp
Console.WriteLine("Welcome to my program")
Console.WriteLine("This is line 2";
Console.Writeline("This is line 3");
```

## 📝 Steps

1. Create a new project:
   ```bash
   dotnet new console -o DebugChallenge
   cd DebugChallenge
   ```

2. Try to type this code into `Program.cs`

3. **Look for errors** - VS Code or Cursor might show red squiggly lines!

4. **Find all 3 mistakes:**
   - Error 1: _______________
   - Error 2: _______________
   - Error 3: _______________

5. **Fix the errors** and run the program: `dotnet run`

6. Verify it works - you should see all three lines printed correctly

## 💡 Hints

- Check punctuation carefully
- Check spelling and capitalization
- Check quotes - do they match?
- Look at each line individually

## 🎓 Learning Goals

By completing this challenge, you'll practice:
- Reading error messages
- Finding common mistakes
- Understanding C# syntax rules
- Debugging skills

## ✅ Success Criteria

Your program is fixed when:
- ✅ It compiles without errors
- ✅ It runs successfully
- ✅ All three lines print correctly
- ✅ You can explain what each error was

## 🔍 Check Your Work

After you've found and fixed the errors, check `debug-challenge-solution.md` to see:
- What each error was
- Why it was wrong
- How to fix it
- Tips for avoiding these mistakes in the future

## 🎉 Bonus Challenge

After fixing the code, try:
- Adding a fourth line
- Changing the messages
- Experimenting with different formatting

**Remember:** Finding and fixing errors is a superpower! Every programmer does this constantly. You're learning a valuable skill! 💪
