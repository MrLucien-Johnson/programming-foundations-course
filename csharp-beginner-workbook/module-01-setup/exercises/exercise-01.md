# Exercise 1: Personal Greeting Program

## 🎯 Your Task

Create a program that greets you and tells you something about yourself.

## 📝 Steps

1. Create a new project (if you haven't already):
   ```bash
   dotnet new console -o PersonalGreeting
   cd PersonalGreeting
   ```

2. Open `Program.cs` in Cursor or VS Code

3. Modify `Program.cs` to print:
   - Your name
   - Your favorite hobby or interest
   - One thing you hope to learn from this course

## 💡 Example Output

```
Hello! I'm Sarah.
I love reading science fiction books.
I hope to learn how to build useful tools with code.
```

## ✅ Tips

- Use `Console.WriteLine()` for each line
- Remember to put text in quotes: `"Your text here"`
- Don't forget the semicolon at the end of each line: `;`
- Save your file (`Ctrl + S` or `Cmd + S`)
- Run your program: `dotnet run`

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Using `Console.WriteLine`
- Writing multiple lines of output
- Making your program personal and meaningful

## 🔍 Check Your Work

Once you've written your program and it runs successfully, check the solution in `exercise-01-solution.md` to see one possible approach.

**Remember:** There are many ways to solve this! If your program runs and shows your information, you've succeeded! 🎉
