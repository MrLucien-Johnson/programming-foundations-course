# Module 1 Exercises

This folder contains all the exercises and their solutions for Module 1.

## 📚 Exercise Files

### Exercises (Try These First!)
- **exercise-01.md** - Personal Greeting Program
- **exercise-02.md** - Multi-Line Message
- **exercise-03.md** - Predict the Output
- **debug-challenge.md** - Find the Errors
- **quiz.md** - Test Your Understanding

### Solutions (Check After You Try)
- **exercise-01-solution.md** - Solution to Exercise 1
- **exercise-02-solution.md** - Solution to Exercise 2
- **exercise-03-solution.md** - Solution to Exercise 3
- **debug-challenge-solution.md** - Solution to the Debug Challenge
- **quiz-answers.md** - Answers to the Quiz

## 🎯 How to Use These Files

1. **Start with the exercise files** (exercise-01.md, exercise-02.md, etc.)
   - Read the task description
   - Try to solve it yourself
   - Write your code
   - Test it with `dotnet run`

2. **Then check the solution files** (exercise-01-solution.md, etc.)
   - Compare your approach
   - Learn from differences
   - See alternative solutions

3. **Important:** Don't peek at solutions until you've tried!

## 💡 Tips for Success

- **Read carefully** - Make sure you understand what's being asked
- **Type code yourself** - Don't copy-paste! Typing helps you learn
- **Test your code** - Run it and see what happens
- **Make mistakes** - Errors teach you how things work
- **Compare solutions** - See different ways to solve the same problem
- **Experiment** - Try variations and improvements

## 🎓 Learning Approach

1. Read the exercise description
2. Think about what you need to do
3. Write your code
4. Test it (`dotnet run`)
5. Fix any errors
6. Compare with the solution
7. Try variations and improvements

## ✅ Remember

- There are many ways to solve problems
- If your code works, you've succeeded!
- Focus on understanding concepts, not memorizing code
- Making mistakes is normal and helpful for learning

Good luck with your learning! 🚀
