# Exercise 3: Predict the Output

## 🎯 Your Task

Look at this code and **predict** what it will print. Then type it and run it to check if you were right!

## 📝 The Code

```csharp
Console.WriteLine("First line");
Console.WriteLine("Second line");
Console.WriteLine("Third line");
Console.WriteLine("First line");
```

## 🤔 Your Predictions

Before running the code, answer these questions:

1. **How many lines will be printed?**
   - Your prediction: _______________

2. **Will "First line" appear twice?**
   - Your prediction: _______________

3. **What order will they appear in?**
   - Your prediction: _______________

## 🚀 Steps

1. Create a new project:
   ```bash
   dotnet new console -o PredictOutput
   cd PredictOutput
   ```

2. Open `Program.cs` and type the code exactly as shown above

3. **Before running:** Write down your predictions

4. Run the program: `dotnet run`

5. Compare the actual output with your predictions

## 💡 What to Look For

- Does the output match your predictions?
- Notice that "First line" appears twice - once at the start and once at the end
- Programs execute from top to bottom, line by line
- Each `Console.WriteLine` creates a new line in the output

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Understanding program execution order
- Predicting program behavior
- Reading code carefully
- Learning from observation
