# Exercise 1 Solution: Personal Greeting Program

## Solution Code

```csharp
Console.WriteLine("Hello! I'm Sarah.");
Console.WriteLine("I love reading science fiction books.");
Console.WriteLine("I hope to learn how to build useful tools with code.");
```

## Explanation

This solution uses three `Console.WriteLine` statements to print three separate lines. Each line contains:
1. A greeting with your name
2. Information about your interests
3. Your learning goals

## Variations

You can customize this in many ways:

```csharp
// More detailed version
Console.WriteLine("Hello! My name is Sarah Johnson.");
Console.WriteLine("I'm interested in technology and problem-solving.");
Console.WriteLine("My favorite hobby is reading science fiction books.");
Console.WriteLine("I'm learning C# to build tools that help people work more efficiently.");
```

## Key Points

- Each `Console.WriteLine` prints on a new line
- You can include any text you want (as long as it's in quotes)
- The order matters - lines print in the order you write them
