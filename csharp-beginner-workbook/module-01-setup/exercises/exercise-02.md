# Exercise 2: Multi-Line Message

## 🎯 Your Task

Create a program that prints a short story or message across multiple lines. Make it work-related!

## 📝 Steps

1. Create a new project (or use an existing one):
   ```bash
   dotnet new console -o MultiLineMessage
   cd MultiLineMessage
   ```

2. Open `Program.cs` in Cursor or VS Code

3. Write a program that:
   - Uses multiple `Console.WriteLine` statements
   - Prints at least 4 lines
   - Is about something work-related (like a daily task list, announcement, or simple report)

## 💡 Example Ideas

### Option 1: Daily Task Reminder
```
Today's Tasks:
1. Review project requirements
2. Attend team meeting at 2 PM
3. Complete Module 1 exercises
4. Plan tomorrow's work
```

### Option 2: Simple Announcement
```
=== Company Announcement ===
All employees are invited to the
quarterly team meeting this Friday.
Time: 3:00 PM
Location: Conference Room A
```

### Option 3: Product Information
```
Product: Wireless Mouse
Price: $29.99
In Stock: Yes
Rating: 4.5 stars
```

## ✅ Tips

- Use one `Console.WriteLine()` per line
- You can use symbols like `===` or `---` for formatting
- Make it realistic - think about what you'd see at work
- Save and run: `dotnet run`

## 🎓 Learning Goals

By completing this exercise, you'll practice:
- Using multiple `Console.WriteLine` statements
- Creating formatted output
- Thinking about real-world applications

## 🔍 Check Your Work

Once you've written your program and it runs successfully, check the solution in `exercise-02-solution.md` to see example solutions.

**Remember:** Your program can be different from the examples - be creative! 🎨
