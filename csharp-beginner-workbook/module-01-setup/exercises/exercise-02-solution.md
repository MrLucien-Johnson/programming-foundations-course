# Exercise 2 Solution: Multi-Line Message

## Solution Code: Daily Task Reminder

```csharp
Console.WriteLine("Today's Tasks:");
Console.WriteLine("1. Review project requirements");
Console.WriteLine("2. Attend team meeting at 2 PM");
Console.WriteLine("3. Complete Module 1 exercises");
Console.WriteLine("4. Plan tomorrow's work");
```

## Alternative Solution: Simple Announcement

```csharp
Console.WriteLine("=== Company Announcement ===");
Console.WriteLine("All employees are invited to the");
Console.WriteLine("quarterly team meeting this Friday.");
Console.WriteLine("Time: 3:00 PM");
Console.WriteLine("Location: Conference Room A");
```

## Alternative Solution: Product Information

```csharp
Console.WriteLine("Product: Wireless Mouse");
Console.WriteLine("Price: $29.99");
Console.WriteLine("In Stock: Yes");
Console.WriteLine("Rating: 4.5 stars");
```

## Explanation

This exercise demonstrates that:
- You can use multiple `Console.WriteLine` statements
- Each statement prints on its own line
- You can create formatted output by planning your lines
- Real-world applications often involve displaying structured information

## Key Points

- Each `Console.WriteLine` creates a new line
- You can create simple formatting with spacing and symbols
- This pattern is useful for displaying lists, announcements, and information
