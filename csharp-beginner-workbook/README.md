# C# Beginner's Workbook: From Zero to Skilled Worker

Welcome! This workbook is designed to take you from having no programming experience to building real, useful programs that you can show to employers. By the end, you'll have practical skills and a portfolio project that demonstrates your abilities.

## 🎯 What You'll Achieve

After completing this workbook, you will be able to:

- Build and understand small C# console applications
- Write code that solves real-world problems
- Debug and fix errors in your programs
- Create a portfolio project you can mention in job applications
- Feel confident using Cursor or Visual Studio Code and the .NET development tools

## 🚀 How to Use This Workbook

1. **Work through modules in order** - Each module builds on the previous one
2. **Type all code yourself** - Don't copy-paste! Typing helps you learn
3. **Try the exercises** - Practice is how you build confidence
4. **Check your answers** - Solutions are provided so you can learn from mistakes
5. **Don't rush** - Take your time and make sure you understand before moving on

## 💻 How to Run Code

### Running C# Examples

```bash
# Navigate to a module folder
cd module-01-setup

# Create a new console project
dotnet new console -o HelloWorld
cd HelloWorld

# Run the program
dotnet run
```

### Running Your Own Code

```bash
# Create a new project
dotnet new console -o MyProject
cd MyProject

# Edit Program.cs in your code editor
# Then run it
dotnet run
```
