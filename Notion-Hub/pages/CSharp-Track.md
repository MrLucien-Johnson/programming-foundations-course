# 💻 C# Track

## Welcome to C# Programming!

This track will teach you C# from absolute zero to building a complete application. No prior experience needed!

---

## 🎯 What You'll Learn

By completing this track, you'll be able to:

- ✅ Write C# programs from scratch
- ✅ Use variables, data types, and user input
- ✅ Make programs that make decisions (if/else)
- ✅ Create loops to repeat tasks
- ✅ Write reusable methods
- ✅ Work with arrays and List<T>
- ✅ Create classes and objects (Object-Oriented Programming)
- ✅ Build a complete Task Tracker application with .NET
- ✅ Debug and fix errors in your code

---

## 📚 Module Overview

### Module 1: Setup & First Program
**What you'll do:**
- Install .NET SDK and your code editor
- Create your first C# console project
- Understand `Console.WriteLine` and basic output
- Learn to run C# programs with `dotnet run`

**Time:** 4-6 hours

**Start here:** [Module 1 →]([MODULE_01_LINK])

---

### Module 2: C# Basics
**What you'll learn:**
- Variables and data types (int, string, double, bool)
- Getting user input with `Console.ReadLine()`
- Type conversion (int.Parse, double.Parse)
- Basic calculations
- String interpolation

**Time:** 4-6 hours

**Start here:** [Module 2 →]([MODULE_02_LINK])

---

### Module 3: Control Flow
**What you'll learn:**
- Making decisions with `if`, `else if`, `else`
- Comparison operators
- Logical operators (`&&`, `||`)
- `for` loops, `while` loops, `foreach` loops
- Creating interactive menus

**Time:** 4-6 hours

**Start here:** [Module 3 →]([MODULE_03_LINK])

---

### Module 4: Methods
**What you'll learn:**
- Defining methods
- Method parameters
- Return values and `void`
- Scope and local variables
- Code organization and reusability

**Time:** 4-6 hours

**Start here:** [Module 4 →]([MODULE_04_LINK])

---

### Module 5: Collections
**What you'll learn:**
- Arrays: creating and accessing elements
- List<T>: flexible collections
- Adding, removing, and counting items
- Looping through collections
- Indexing and common operations

**Time:** 4-6 hours

**Start here:** [Module 5 →]([MODULE_05_LINK])

---

### Module 6: Intro to Object-Oriented Programming
**What you'll learn:**
- Classes and objects
- Properties and fields
- Constructors
- Methods within classes
- Creating and using objects

**Time:** 4-6 hours

**Start here:** [Module 6 →]([MODULE_06_LINK])

---

### Module 7: Task Tracker Project
**What you'll build:**
- A complete .NET console-based Task Tracker application
- Multi-file project structure
- TaskItem and TaskManager classes
- Interactive menu system
- Portfolio-ready project!

**Time:** 6-8 hours

**Start here:** [Module 7 →]([MODULE_07_LINK])

## 💡 Tips for Success

### Do's ✅

- **Complete modules in order** - Concepts build on each other
- **Type all code yourself** - Don't copy-paste!
- **Run examples** - See code in action
- **Do all exercises** - Practice is essential
- **Check solutions** - After you've tried first
- **Take notes** - Write down what you learn

### Don'ts ❌

- **Don't skip modules** - Each builds on the previous
- **Don't rush** - Understanding > Speed
- **Don't just read** - You must write code
- **Don't give up** - Getting stuck is normal
