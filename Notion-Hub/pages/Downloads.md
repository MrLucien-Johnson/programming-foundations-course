## 🖥️ How to Download and Open

### Step 1: Download

1. Click the download link for your chosen workbook
2. Wait for download to complete
3. Find the ZIP file in your Downloads folder

### Step 2: Extract/Unzip

**Windows:**
1. Right-click the ZIP file
2. Select "Extract All..."
3. Choose a location (e.g., Desktop or Documents)
4. Click "Extract"

**Mac:**
1. Double-click the ZIP file
2. It will automatically extract to the same location

**Linux:**
```bash
unzip workbook-name.zip
```

### Step 3: Open Your Workbook

1. Navigate to the extracted folder
2. Open `README.md` to start
3. Or go directly to `module-01-setup/` to begin Module 1

## 💻 Opening Files

### Markdown Files (.md)

**Recommended editors:**
- **VS Code** - Free, great for coding
- **Cursor** - VS Code-based editor
- **Notepad++** (Windows) - Simple text editor
- **TextEdit** (Mac) - Built-in editor

**How to open:**
- Double-click the `.md` file
- Or right-click → "Open with" → Choose your editor

### Code Files (.py, .cs)

**Python files (.py):**
- Open with VS Code, Cursor, or any text editor
- Run with: `python filename.py`

**C# files (.cs):**
- Open with VS Code, Cursor, or Visual Studio
- Run with: `dotnet run` (in project folder)

## ❓ Troubleshooting Downloads

### ZIP File Won't Open?

**Windows:**
- Right-click → "Extract All..."
- Or use 7-Zip (free download)

**Mac:**
- Double-click should work
- Or use The Unarchiver (free from App Store)

**Linux:**
```bash
unzip filename.zip
```

### Files Look Weird?

- Make sure you're opening `.md` files with a text editor
- VS Code or Cursor will display markdown nicely
- Don't try to open code files in Word or Pages

### Can't Find Files?

- Check your Downloads folder
- Search for the workbook name
- Make sure extraction completed successfully
