# AI Prompt Creation Track (Beginner Friendly)

Welcome! This track teaches you how to design prompts clearly and safely, even if you are brand new
to AI. You will learn how to turn vague ideas into clear instructions, test your results, and build
simple, reliable prompt workflows.

## ✅ Who this is for
- Absolute beginners who want to learn prompt writing step by step
- Anyone who wants a clear, repeatable process (not “guess and hope”)

## 🎯 What you’ll build
- A prompt playbook you can reuse
- A mini evaluation set to test quality and consistency
- A simple workflow that is safe and repeatable

## 📚 Modules (follow in order)
1. **AI Foundations**  
   Learn how LLMs work in simple terms and how to write clear prompt goals.  
   → [Start Module 1](../languages/ai/beginner/modules/01-ai-foundations.md)

2. **Prompting Basics**  
   Build strong prompts using structure, tone, and constraints.  
   → [Start Module 2](../languages/ai/beginner/modules/02-prompting-basics.md)

3. **Prompt Patterns**  
   Learn the most useful prompt patterns for real tasks.  
   → [Start Module 3](../languages/ai/beginner/modules/03-prompt-patterns.md)

4. **Evaluation & Iteration**  
   Test prompts, track changes, and improve results step by step.  
   → [Start Module 4](../languages/ai/beginner/modules/04-evaluation-and-iteration.md)

5. **Safety & Policy Basics**  
   Avoid risky outputs and build safe prompt behavior.  
   → [Start Module 5](../languages/ai/beginner/modules/05-safety-and-policy-basics.md)

6. **Workflows & Automation**  
   Combine prompts into simple workflows you can reuse.  
   → [Start Module 6](../languages/ai/beginner/modules/06-workflows-and-automation.md)
