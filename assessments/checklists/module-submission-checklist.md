# Module Submission Checklist

Use this for module-level exercises and practice work.

- [ ] Branch name matches `submission/<learner>/<language>/<module>`.
- [ ] Work is placed under `submissions/<learner>/<language>/<module>/`.
- [ ] All required exercises are completed.
- [ ] Code runs without errors.
- [ ] A short `README.md` explains setup and run steps.
- [ ] Test results are included if tests exist.
- [ ] Self-check against `assessments/rubrics/code-review-rubric.md`.
