## 🗺️ Course Structure Overview

### Each Track Has 7 Modules:

1. **Setup & First Program** - Get everything working
2. **Basics** - Variables, input, calculations
3. **Control Flow** - Decisions and loops
4. **Functions/Methods** - Reusable code blocks
5. **Collections** - Lists, arrays, dictionaries
6. **Object-Oriented Programming** - Classes and objects
7. **Portfolio Project** - Build the Task Tracker!

### Each Module Contains:

- 📖 **Full lesson** - Clear explanations, no jargon
- 💡 **Examples** - Working code you can run
- ✏️ **Exercises** - Practice problems to solve
- ✅ **Solutions** - Answers to check your work
- 🐛 **Debug challenges** - Find and fix errors
- 📝 **Quizzes** - Test your understanding

## 💻 Running Code

### For Python:

```bash
# Navigate to example folder
cd python-beginner-workbook/module-01-setup/examples

# Run a Python file
python example-01-basic.py
```

### For C#:

```bash
# Navigate to module folder
cd csharp-beginner-workbook/module-01-setup

# Create a new project (if needed)
dotnet new console -o HelloWorld
cd HelloWorld

# Run the program
dotnet run
```

**Detailed setup instructions are in Module 1 of each workbook.**
