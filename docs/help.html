Quick answers for new learners

These answers are written for people who are new to computers.

I do not know where my download went.

Look in your Downloads folder. On most computers, you can open Downloads from your file manager.

What is a ZIP file?

A ZIP is a folder that is compressed to save space. Double-click it to open it, then open the folder inside.

How do I open a README file?

A README is a text file. Double-click it and it will open. If it does not, right-click and choose “Open with” then pick a text editor.

Do I need to use Git or the command line?

No. Download the ZIP and follow the steps. The course is made for beginners.
