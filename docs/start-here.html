Your first steps

Follow these steps in order. If you get stuck, open the Help page. You do not need to know any programming to start.

Download the course

Click the download button. This saves the course to your computer so you can open the lessons like normal files.

Download course ZIP

Open the folder

Find the ZIP file you downloaded, open it, and look for a folder named programming-foundations-course. Open that folder.

Choose a course

Pick one course to start. Python is recommended for absolute beginners. C# is great if you want Microsoft or game development. The AI Prompt course teaches prompt writing from beginner to advanced.

Pick a course AI Start Here

Open Module 1

Inside your chosen course, open Module 1. You can read it online here, or open it from the ZIP folder you downloaded.

Open Python Module 1 Open C# Module 1 Open AI Module 1

If you downloaded the ZIP, open python-beginner-workbook/README.md or csharp-beginner-workbook/README.md.

Prefer to preview online?

You can also read the files in your browser first. When you are ready to code, download the ZIP and open the files on your computer.
